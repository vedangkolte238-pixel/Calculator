🧮 Responsive Calculator

A modern and responsive calculator built using **HTML**, **CSS**, and **JavaScript**. The calculator supports basic arithmetic operations, percentage calculations, brackets, decimal numbers, and includes a calculation history panel for tracking previous results. Designed with a clean and intuitive interface, it works smoothly across desktop and mobile devices.

✨ Features

* Modern and responsive design
* Clean and user-friendly interface
* Basic arithmetic operations

  * Addition (+)
  * Subtraction (−)
  * Multiplication (×)
  * Division (÷)
* Percentage (%) calculations
* Bracket support `( )`
* Decimal number support
* Clear (C) button
* Backspace (⌫) button
* Calculation history panel
* Clear history option
* Fast and accurate calculations
* Error handling for invalid expressions

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

🚀 Getting Started

1. Clone or download this repository.
2. Open the project folder.
3. Open `calc.html` in your preferred web browser or run it using Live Server in Visual Studio Code.

📖 How to Use

* Click the number and operator buttons to create an expression.
* Press **=** to calculate the result.
* Use **C** to clear the display.
* Use **⌫** to remove the last entered character.
* View previous calculations in the **History** panel.
* Click **Clear** in the history section to remove all saved calculations.

🎯 Purpose

This project was created to improve front-end web development skills by implementing calculator logic, DOM manipulation, event handling, and responsive user interface design using JavaScript.
